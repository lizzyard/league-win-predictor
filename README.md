# League Win Predictor

## Overview

Predicts the Probability of winning a League of Legends match using 15-minute game statistics

## Technologies

- Python 
- Pands
- Scikit-learn 
- Streamlit
- Riot Games API
- Git

## Machine Learning Models

- Logistic Regression
- Decision Tree
- Random Forest

## Features

- Gold Difference
- XP Difference
- CS Difference

## Results

Best Model: Logistic Regression
Accuracy: 60%
